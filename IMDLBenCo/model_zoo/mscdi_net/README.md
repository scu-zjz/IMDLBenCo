## Pretrained Weights

MSCDI-Net requires the pretrained SegFormer weights. Please download the `mit_b4` checkpoint from the [official SegFormer repository](https://github.com/NVlabs/SegFormer).

Since the official download link is no longer accessible, an alternative download link is provided below:

https://www.123865.com/s/NesVjv-In4hv

After downloading, please place the pretrained weight file in the appropriate directory specified by the configuration file or the script.
